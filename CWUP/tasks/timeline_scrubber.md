# Timeline Scrubber shows frame number
Display the frame like Blender is doing.
Use accent color from default theme declared in ForgeRunner/theme.sml.
Also use vertical line like in picture.
A dark line each 10 frames and a gray line in between, 5,15, 25,...

Also the frame numbers above the line shall be centered to the line, like in the image.
